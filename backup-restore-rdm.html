Backup and Restore

Cara Backup / Restore RDM - Rapor Digital Madrasah

  • Hosting RDM
  • Mei 10, 2024

RDM perlu dibackup secara berkala, minimal setelah semua nilai selesai. Hal ini dilakukan agar ketika Hosting mengalami masalah, Anda punya backup Database RDM.

Hal ini juga dilakukan ketika Anda pindah Hosting. Karena pihak hosting hanya menjaga keamanan web, selanjutnya untuk Backup Database RDM adalah tanggung jawab Proktor Madrasah.

Cara Backup RDM

Untuk membackup RDM silahkan ikuti langkah-langkah berikut ini:

  1. Masuk ke RDM sebagai Proktor
  2. Klik Backup Restore, seperti gambar di bawah

  3. Klik Backup DB atau Backup Semua (boleh mana saja), kemudian tunggu beberapa saat.

  4. Klik Download untuk menyimpan di Komputer. Perhatikan Folder tempat penyimpanan, sehingga memudahkan ketika restore.

Cara Restore DataBase RDM

Sama seperti langkah Download, di atas.
  1. Masuk ke RDM sebagai Proktor
  2. Klik Restore, seperti gambar di bawah

  3. Pilih lokasi penyimpanan Database yang sudah didownload sebelumnya.
  4. Tunggu sampai proses Restore Selesai.
Semoga bermanfaat
