Terms and Conditions

1. Siapa Kami?

Kami bukan bagian dari KEMENAG. Kami adalah layanan hosting yang bertujuan untuk mempermudah Proktor dalam installasi RDM. Kami tidak menjual Aplikasi RDM, karena itu adalah hak Kemenag RI dan didistribusikan secara gratis. Kami menggunakan server yang handal, dan akan membatasi jumlah sub domain. Apabila customer bertambah, kami akan menambah kapasitas server sehingga akan tetap nyaman digunakan.

2. Data apa yang kami kumpulkan tentang Anda?

Sebagai Hosting RDM tentunya ada beberapa informasi yang kami kumpulkan tentang Anda. Data anda sangat aman, kami hanya mengumpulkan data Anda untuk kepentingan database RDM. Penggunaan Versi VDI, XAMPP ataupun Hosting pasti akan mengumpulkan data yang diperlukan.
  • NUPTK akan tersimpan pada database RDM
  • NPK akan tersimpan pada database RDM
  • Data Guru akan tersimpan pada database RDM
  • Data Siswa akan tersimpan pada database RDM
  • Nilai Siswa akan tersimpan pada database RDM
Data Anda Aman Bersama Kami
Kami tidak akan pernah masuk ke database anda. Kami hanya melakukan backup secara rutin untuk kemanan Web. Adapun untuk Data RDM silahkan Proktor melakukan backup untuk kepentingan sendiri.

3. Cookies dan teknologi sejenisnya

Kami menggunakan cookie untuk mengelola sesi Anda. "Cookies" adalah file teks kecil yang ditransfer oleh server web ke hard drive perangkat Anda. Cookie dapat digunakan untuk mengumpulkan tanggal dan waktu kunjungan Anda, riwayat penelusuran Anda, preferensi Anda, dan nama pengguna Anda. Anda dapat mengatur browser Anda untuk menolak semua atau beberapa cookie, atau untuk mengingatkan Anda ketika situs web mengatur atau mengakses cookie. Jika Anda menonaktifkan atau menolak cookie, harap perhatikan bahwa beberapa bagian dari Layanan kami mungkin menjadi tidak dapat diakses atau tidak berfungsi dengan baik.
